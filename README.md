FinFit: Get Financially Fit

Overview
This repository hosts the official landing page for FinFit: Get Financially Fit, a financial literacy event organized by the Fintech Club and IIC at VIT Bhopal.
The website provides all necessary information about the event, including the speaker profile, agenda, registration details, and pricing tiers.
